# QC课题申报书：构建“中国卷烟博物馆”——一个安全、合规、有深度的烟草知识平台

## 一、课题背景与意义

当前，中国烟草市场信息呈现碎片化状态，缺乏一个集中、准确、且符合监管要求的数字化知识平台。现有平台如烟悦网、烟草市场等，或侧重社区互动，或聚焦行业资讯，在**数据的系统性、纯粹性及文化深度**上存在不足。

本项目“中国卷烟博物馆”旨在打造一个**集产品数据库、历史文化展示与智能服务于一体的综合性平台**。我们不仅系统整理3200余种烟草产品的规格、价格等核心数据，更将深度融入品牌及制造商的发展历史，并规划了“真伪鉴别指南”、“雪茄保存贴士”、“产品智能对比”及“AI个性化推荐”等实用功能。平台坚持**纯粹、专业、无广告**的原则，严格遵守国家法律法规，致力于成为烟草知识科普、文化传承与智能服务的权威窗口。

## 二、课题目标

1.  **构建核心数据库**：建立并维护一个包含3200+ SKU的中国烟草产品（卷烟、雪茄、电子烟）在线数据库，确保数据准确、结构清晰。
2.  **深化文化内容**：系统梳理并呈现重点烟草品牌及制造商的发展历程、文化故事，丰富平台的知识维度。
3.  **拓展实用功能**：
    *   **知识科普**：上线“真伪鉴别指南”、“雪茄保存贴士”等板块。
    *   **智能服务**：开发“产品对比”工具和基于用户偏好的“AI推荐”功能。
4.  **确保安全合规**：
    *   严格遵守《广告法》及烟草相关法规，所有内容以**知识科普**为导向。
    *   实施**年龄验证机制**，防止未成年人访问。
    *   数据来源优先采用官方渠道，并附带免责声明。
5.  **优化用户体验**：提供高效、直观的搜索、筛选、对比与推荐功能，打造稳定、快速的现代化Web应用。

## 三、风险分析与应对策略

| 风险点 | 应对策略 |
| :--- | :--- |
| **政策合规风险** | 平台定位为非商业性的知识科普与文化教育；显著位置标注“吸烟有害健康”警示语；杜绝任何诱导性内容。 |
| **数据准确性风险** | 优先采用国家局、中烟公司等官方信源；建立定期核查更新机制；明确标注“数据仅供参考”。 |
| **未成年人访问风险** | 首次访问强制进行年龄确认（18+）；通过`robots.txt`限制搜索引擎索引。 |
| **AI竞品竞争风险** | 聚焦**深度、系统性与智能化**，提供AI快问快答无法替代的结构化数据、历史文化沉淀及个性化智能服务；强化社区价值与纯粹无干扰的用户体验。 |

## 四、实施方案

1.  **核心数据与内容建设（一期）**：
    *   制定标准化数据模板，涵盖卷烟产品的规格、官方定价、厂商信息及高清图片。
    *   启动“品牌/制造商历史”专题内容创作，挖掘文化价值。
2.  **平台开发与安全加固**：
    *   迭代优化现有Web应用（https://tobacco-atlas.vercel.app/），集成年龄验证等安全模块。
    *   完善数据管理后台，确保内容更新的规范与高效。
3.  **合规审查与社区引导**：
    *   建立内容发布前的合规审查流程。
    *   制定社区规则，引导用户围绕知识分享与文化探讨进行高质量互动。
4.  **未来功能拓展（二期及三期）**：
    *   **扩展雪茄品类**：将数据库覆盖范围延伸至雪茄烟，并建立市场零售价格的动态采集与更新机制。
    *   **深化科普教育**：增加“真伪鉴别指南”、“雪茄保存贴士”等实用知识板块。
    *   **引入智能服务**：开发多维度“产品对比”功能，并基于用户行为数据实现安全、合规的“AI个性化推荐”。

## 五、预期效果

1.  **建成一个安全、合规、权威的线上烟草知识库**，有效填补市场空白。
2.  **形成独特的“数据+文化+科普+智能”四位一体模式**，在与AI助手等竞品的竞争中建立全面的差异化优势。
3.  **为烟草爱好者、研究者及行业从业者提供一个纯净、专业、智能的信息服务平台**，兼具实用价值、社会文化价值与前瞻性。
4.  **通过真伪鉴别、保存贴士、产品对比及AI推荐等系列功能，全方位提升用户的知识水平、决策效率和消费体验**，巩固平台作为烟草领域知识与服务核心阵地的地位。
