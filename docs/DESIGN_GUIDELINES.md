# 🎨 Tobacco-Atlas Web 设计指南

## 设计原则

### 1. 简洁直观
- 界面元素精简，避免信息过载
- 重要的数据和功能优先展示
- 保持一致的视觉层次

### 2. 数据驱动
- 烟草品牌、制造商、产品数据为核心
- 图表和可视化帮助理解数据关系
- 支持数据筛选和搜索

### 3. 响应式设计
- 适配桌面、平板、手机等不同设备
- 移动端优先的交互体验
- 触摸友好的组件尺寸

### 4. 专业可信
- 使用专业的色彩搭配（深蓝、灰色、白色为主）
- 字体选择清晰易读
- 保持政府/执法相关的严肃性

## 色彩系统

### 主色调
- **Primary**: `#1e40af` (深蓝色) - 代表权威和专业
- **Secondary**: `#64748b` (石板灰) - 辅助色
- **Accent**: `#dc2626` (红色) - 警示和重要操作

### 状态色
- **Success**: `#16a34a` (绿色)
- **Warning**: `#d97706` (橙色)  
- **Error**: `#dc2626` (红色)
- **Info**: `#0ea5e9` (蓝色)

## 字体系统

- **主要字体**: Inter, system-ui, sans-serif
- **标题**: 600-700 字重
- **正文**: 400-500 字重
- **代码**: monospace

## 组件规范

### 导航栏 (Navbar)
- 固定顶部
- 包含 Logo、主要导航链接、用户操作
- 移动端折叠为汉堡菜单

### 产品卡片 (ProductCard)
- 统一的卡片布局
- 包含品牌 Logo、产品图片、基本信息
- 悬停效果显示更多详情

### 数据表格 (DataTable)
- 支持排序、筛选、分页
- 行悬停高亮
- 批量操作支持

### 表单组件 (Form)
- 清晰的标签和占位符
- 实时验证反馈
- 错误状态明显标识

## 页面布局

### 首页 (Home)
- 英雄区域介绍项目
- 快速搜索和筛选
- 数据统计概览

### 品牌列表 (BrandList)
- 网格布局展示品牌
- 支持按字母、国家、类型筛选
- 点击进入品牌详情

### 制造商列表 (ManufacturerList)
- 类似品牌列表的布局
- 显示制造商基本信息
- 关联品牌展示

### 管理后台 (AdminDashboard)
- 侧边栏导航
- 数据提交表单
- 用户管理功能

## 交互规范

### 加载状态
- 骨架屏用于数据加载
- 按钮加载状态显示进度
- 错误状态提供重试选项

### 反馈机制
- 成功操作显示 Toast 通知
- 错误信息具体且可操作
- 确认对话框用于重要操作

### 动画效果
- 微交互动画增强用户体验
- 页面过渡平滑
- 避免过度动画影响性能

## 可访问性

- 符合 WCAG 2.1 AA 标准
- 键盘导航支持
- 屏幕阅读器兼容
- 足够的对比度

## 技术栈约定

- **框架**: React 18 + TypeScript
- **样式**: Tailwind CSS + shadcn/ui
- **状态管理**: React Context + useReducer
- **路由**: React Router v6
- **测试**: Vitest + React Testing Library

## 版本控制

- 功能分支命名: `feature/xxx`
- 修复分支命名: `fix/xxx`  
- 文档分支命名: `docs/xxx`
- 提交信息格式: `type(scope): description`
