# 中国卷烟博物馆开发计划 (Tobacco Atlas Development Plan)

本文档基于项目PRD和当前状态，制定了清晰、可执行的分阶段开发计划，旨在高效、有序地推进项目落地。

## 一、项目当前状态

### 已完成
- **项目规划与文档**：PRD、QC课题申报书（中英文）已完善。
- **基础资产**：UI/UX设计稿、Vercel部署 (`tobacco-atlas.vercel.app`) 已就绪。

### 待完成
- **前端应用开发**：核心功能代码尚未完全实现。
- **数据层建设**：3200+ SKU的结构化产品数据库尚未建立。
- **高级功能**：产品对比、AI推荐、真伪鉴别、雪茄保存贴士等功能待开发。
- **多语言支持**：英文版网站未上线。

## 二、分阶段开发计划

### 阶段一：夯实基础 (MVP - 2-3周)
**目标**：构建一个功能完整、数据准确的最小可行产品。
- **[ ] 搭建前端框架**：使用Next.js 14 (App Router) + TypeScript + Tailwind CSS 初始化项目，并集成PRD中的设计系统。
- **[ ] 构建数据管道**：创建标准化JSON数据结构，录入首批 **100+ 核心卷烟产品** 数据。
- **[ ] 实现MVP功能**：完成产品目录、产品详情页、基础搜索和筛选功能。
- **[ ] 安全合规**：集成年龄验证弹窗和健康警示语。

### 阶段二：内容扩展与功能深化 (v1.1 - 3-4周)
**目标**：丰富内容广度，增加核心交互功能。
- **[ ] 扩充数据库**：将产品库扩展至 **500+ SKU**，覆盖全部卷烟、首批雪茄和电子烟。
- **[ ] 开发核心工具**：实现 **“产品对比”** 功能和 **“收藏夹”** 功能。
- **[ ] 上线知识板块**：创建并填充 **“真伪鉴别指南”** 和 **“雪茄保存贴士”** 页面。
- **[ ] 启动文化内容**：开始撰写首批 **品牌/制造商历史** 文章。

### 阶段三：智能化与国际化 (v2.0 - 4-6周)
**目标**：引入智能服务，完成国际化，提升平台竞争力。
- **[ ] 实现AI推荐**：基于用户行为，开发安全、合规的个性化推荐功能。
- **[ ] 完成多语言**：实现网站的 **中英双语切换**。
- **[ ] 完善数据**：将数据库扩充至 **3200+ SKU** 的目标规模，并建立动态更新机制。
- **[ ] 社区与SEO**：优化SEO，为未来社区功能打下基础。
