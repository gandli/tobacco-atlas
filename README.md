# Tobacco Atlas

> A comprehensive guide to Chinese tobacco products: cigarettes, cigars, and e-cigarettes.

## Overview

Tobacco Atlas is an informational website inspired by [ciggies.app](https://www.ciggies.app/), designed to provide comprehensive information about tobacco products available in China.

### Product Categories

- **Cigarettes (香烟)** - Traditional Chinese cigarette brands and varieties
- **Cigars (雪茄烟)** - Domestic and imported cigar products
- **E-cigarettes (电子烟)** - Vaping devices and e-liquid brands

## Features

- 🎨 Clean, visual-first design similar to ciggies.app
- 📱 Mobile-responsive layout
- 🔍 Search and filter by brand, type, price range
- 📊 Product specifications and comparison tools
- 🌐 Bilingual support (Chinese/English)

## Tech Stack

- Frontend: React / Next.js
- Styling: Tailwind CSS
- Data: Static JSON / Markdown content
- Deployment: Vercel / GitHub Pages

## Getting Started

```bash
# Clone the repository
git clone https://github.com/gandli/tobacco-atlas.git

# Install dependencies
npm install

# Run development server
npm run dev
```
