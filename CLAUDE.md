# Design Context

## 项目概述
**中国烟草图谱 (Chinese Tobacco Atlas)** 是一个数字博物馆，展示中国香烟品牌和产品。复刻参考网站：https://www.ciggies.app/

## 用户
- **目标用户**: 卷烟收藏爱好者、对烟草文化感兴趣的研究者、寻找特定品牌信息的消费者
- **使用场景**: 浏览产品目录、查询品牌信息、查看产品规格参数、社区互动
- **核心需求**: 快速获取产品信息、收藏管理、社区交流、多语言支持（中英双语）

## 品牌个性
- **档案感**: 强调数据库的完整性和历史档案价值
- **专业**: 详细的产品规格参数、准确的分类体系
- **精致**: 高端的视觉体验，类似 ciggies.app 的设计质感

## 设计方向
- **参考设计**: ciggies.app - 深色背景、卡片式布局、档案感
- **视觉风格**: 现代简洁、信息密度高、图片为主
- **主题支持**: 明暗双主题（当前实现）
- **配色**: 
  - 主色调：中性色（黑/白/灰）
  - 强调色：金色 (#c5a059) 用于高端感
  - 背景：浅色模式纯白，深色模式深蓝灰

## 设计原则
1. **内容优先**: 产品图片是核心，UI 应低调不抢眼
2. **信息层次**: 通过字体大小、颜色深浅建立清晰的视觉层次
3. **一致性**: 所有卡片、按钮、间距遵循统一的设计系统
4. **响应式**: 移动端优先，确保在小屏幕上的可用性
5. **无障碍**: 遵循 WCAG 2.1 AA 标准，确保键盘导航和屏幕阅读器支持

## 技术栈
- **框架**: Next.js 16 (App Router)
- **UI 库**: React 18 + shadcn/ui + Radix UI
- **样式**: Tailwind CSS 3.4
- **图标**: Lucide React
- **国际化**: i18next + react-i18next
- **主题**: next-themes (明暗切换)

## 设计 Token

### 颜色系统 (HSL)
```css
/* 浅色主题 */
--background: 0 0% 100%
--foreground: 0 0% 8%
--primary: 0 0% 8%
--gold: #c5a059 (premium 强调色)
--ash: #0b0b0d (深色背景)

/* 深色主题 */
--background: 222.2 84% 4.9%
--foreground: 210 40% 98%
--primary: 210 40% 98%
```

### 字体
- **无衬线**: "DM Sans", "Inter", system-ui, sans-serif
- **衬线**: "Noto Serif SC", serif (用于标题/展示)

### 圆角
- sm: 0.25rem (4px) - 小元素
- DEFAULT: 0.5rem (8px) - 默认
- md: 0.75rem (12px) - 按钮、输入框
- lg: 1rem (16px) - 卡片
- xl: 1.25rem (20px) - 模态框
- 2xl: 1.5rem (24px) - 大卡片
- 3xl: 2rem (32px) - 特殊元素
- full: 9999px - 圆形（头像）

### 间距系统
使用 Tailwind 默认间距 + 自定义尺寸：
- 字体：2xs(10px), 3xs(9px), 11(11px), 13(13px), 15(15px), 17(17px), 36(36px)
- 宽度：36(144px), 140(140px), 180(180px)
- 高度：180(180px), 260(260px)
- 最大宽度：300(300px), 380(380px), 420(420px), 1200(1200px), 1600(1600px)

## 组件规范

### 导航栏
- 高度：`var(--nav-height)` (3.5rem)
- 背景：`bg-background/80` + `backdrop-blur-md`
- 边框：`border-border/50`
- 固定定位：`fixed top-0`

### 产品卡片 (SKU Card)
- 悬停效果：渐变遮罩 + 信息上浮动画
- 圆角：28px (特殊设计)
- 阴影：`shadow-[0_10px_30px_rgba(15,23,42,0.16)]`

### 博物馆风格组件
- `.museum-inline-panel`: 圆角 2xl + 半透明背景
- `.museum-inline-chip`: 圆角 xl + 半透明背景

## 无障碍要求
- WCAG 2.1 AA 对比度标准
- 所有交互元素支持键盘导航
- 图片提供 alt 文本
- 表单元素关联 label
- 焦点状态清晰可见

## 图标使用
- 主图标库：Lucide React (与 shadcn/ui 一致)
- 风格：outline 线框为主
- 语义化使用：确保图标有明确的含义
